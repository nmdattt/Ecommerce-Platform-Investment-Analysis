# Project - Phát triển bán hàng trên app mobile hay website

## Mô tả
Dự án này phân tích bộ dữ liệu Ecommerce_Customers.csv về hành vi khách hàng khi mua quần áo trực tuyến, nhằm giúp công ty quyết định nên tập trung phát triển nền tảng mobile app hay website.

## Nội dung phân tích

1. **Bảng tóm tắt, khái quát về dữ liệu.**
2. **Thời gian tương tác trên 2 nền tảng có khác biệt không?**
   - Phân tích sâu theo chiều dài thời gian là khách hàng thành viên.
3. **Lựa chọn biến Y để xây dựng mô hình cho thấy nên đầu tư vào nền tảng mobile app hay website.**
4. **Viết các nhận xét và kết luận.**

## File dự án

## Cách sử dụng

1. Clone repository này về máy.
2. Mở file `.Rmd` bằng RStudio và chạy từng khối lệnh để xem kết quả phân tích.
3. Đảm bảo đã cài đặt các package: `tidyverse`, `readr`, `janitor`, `VIM`, `ggplot2`, `boot`, `leaps`, `caret`, `mgcv`, `mgcViz`, `knitr`, ...
